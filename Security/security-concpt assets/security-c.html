Security Concepts

Tasked to make a secure IAM user that requires formal logging in, created from the main user.

1 Step 1

I went to “User groups” on the IAM service to create a user group.

2 Step 2

I added “SupportEngineers” ln the user group name.

3 Step 3

I attached a permissions policy called “AmazonEC2ReadOnlyAccess” to the user group.

4 Step 4

The user group has been added successfully.

5 Step 5

I went back to IAM on Access management’s, Users, to create a user.

6 Step 6

I plotted my specific user details including the user name of the IAM user and its curated password.

7 Step 7

I set permissions to “Add user to group” with the user being “SupportEngineers” as the main user.

8 Step 8

The review tags were configured as; key being “job-title” and value as “Support Engineers”.

9 Step 9

The user was created successfully.
